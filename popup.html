How It Works?

This extension will try to automatically detect, convert and replace any time shown on the webpages you visit with your local time. See examples below.



Here are a few more links to try it out immediately:
Apple WWDC Live Event site
Apple Music Launch Article on Techcrunch
Oscar dates 2016

Currently the extension support mostly US time conversion, but more will be added soon. Also we plan to add an option to show back the original time when you click on the replaced time. Please expect common updates of the extensions allowing for it improvement soon.

This extension does not add any icons to your browser. Go to Chrome Options - Tools - Extensions menu and find Local Time.

Permissions

This extension will access the websites that you visit in order to modify the displayed time. It also requires access to open a new tab with these instructions, only once, after your first installation. No other actions are performed, nor impacted by this extension.
